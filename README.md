## Overview

**Clinica Rajem** is the official web application and digital presence for a physiotherapy clinic highly specialized in pelvic floor health and rehabilitation. The platform is designed to provide patients with clear, accessible information regarding specialized treatments, clinical approaches, and facility details.

The project utilizes a decoupled architecture, leveraging a robust Java Spring Boot backend to handle server-side logic and routing, paired with a lightweight, highly responsive frontend built with native web technologies.

## Getting Started (Developers)

To run this project locally for development or testing purposes, follow the steps below.

### Prerequisites

Ensure your local development environment meets the following requirements:
* **Java Development Kit (JDK):** Version 11 or higher (depending on your specific Spring Boot configuration).
* **Build Tool:** Maven or Gradle (included via wrapper in standard Spring Boot projects).
* **Node.js & npm:** (Optional, if utilizing a specific SCSS compiler workflow).

### Installation and Execution

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/jclot/ClinicaRajem.git](https://github.com/jclot/ClinicaRajem.git)
   cd ClinicaRajem
   ```

2. **Compile and run the Spring Boot application:**
   If using Maven:
   ```bash
   ./mvnw spring-boot:run
   ```
   If using Gradle:
   ```bash
   ./gradlew bootRun
   ```

3. **Access the application:**
   Open your web browser and navigate to `http://localhost:8080`.
